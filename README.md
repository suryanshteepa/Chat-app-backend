# 💬 Chat App Backend (CRUD)

## 🚀 Features

* Create new chats
* View all chats
* Edit messages
* Delete chats
* Timestamp for each message

## 🛠 Tech Stack

* Node.js
* Express.js
* MongoDB / SQL
* EJS

## 📦 Run Locally

git clone https://github.com/your-username/chat-app-backend.git
cd chat-app-backend
npm install
npm start

## 📌 Description

This is a backend chat application that allows users to perform CRUD operations on messages. It demonstrates server-side development using Node.js and Express.
